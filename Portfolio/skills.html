HTML

Das Konstrukt einer jeden Internetseite. HTML ist der Inhaltder Seite und liefert jegliche Elemente. Dafür ist HTML statisch und Unflexibel.

WORD

Das Programm zum Texte schreiben. Lebensläufe verfassen und so weiter. Wordmist ein kostenpflichtiges Programm von Microsoft, womit man alles mögliche Schreiben kann, was man will und wie man will. Dazu kann man verschiedenste Formen oder sogar Bilder und Videos eifügen und sie entsprechend anpassen.

CSS

Die Anforderungen von HTML. CSS entscheidet, wie eine Seite Aussehen soll. HTML Baut, CSS Designt. CSS kann aber auch Interaktiv sein und einfache Animationen abspielen, ist aber dafür bei der Interaktivität eingeschränkt.

POWERPOINT

Das Vortrag Programm. Gerne mal wird gesagt, dass man eine PowerPoint machen soll, aber was ist das. PowerPoint ist ein Programm von Microsoft sowie Word und Excel udn dient dazu meist Vorträge vorzubereiten, aber man kann auch z.B. ein MockUp machen, was wichtig in der Programmier-Branche ist.

JAVASCRIPT

Das Leben in der Seite. Was bringt es eine Vollkommene Seite ohne Leben zu Kreieren. JavaScript ist schwieriger als HTML oder CSS, kann aber dafür jegliche Interaktionen durchführen, von Textveränderungen bis zum Wortwörtlichem Leben auf der Seite.

EXCLE

Excel. Tabellen, Rechnen, Vergleichen, den PC für einen Arbeiten lassen. Das Programm von Microsoft, Excel. Eyxel wird oft in Büro Jobs verwendet, um z.B. Aufgaben auszurechnen und Aufträge einfacher auszuführen. Aber man kann auch andere Sachen mit Excel machen, besonders als Schüler ist Excel sehr gut geeignet, Inhaltsverzeichnisse zu machen oder derartiges.

MATERIAL DESIGN LITE

Eine Einfache Internetseite worauf man verschiedenste Elemente für alle 3 Bereiche des Codens finden kann, HTML über CSS bis hin zu JS. Einfach nur ein kleiner Link und schon gehts los.

BLENDER

Ein Programm, was Gratis, aber Vielfältig ist. Es braucht nur Kreativität, um gute 3D Modelle zu Modellieren, von Winzigen Ameisen-Modelle bis hin zu Unvorstellbaren Objekten, man braucht nur den richtigen 3D Drucker 😁.
